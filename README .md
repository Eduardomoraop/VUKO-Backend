# VUKO.AI - Proyecto Final de Máster 🚀

Plataforma integral con Dashboard de gestión y servicios de inteligencia artificial.

## 🔗 Enlaces del Proyecto

Demo en Vivo (Frontend):**[https://vuko-backend.vercel.app] (vuko-backend.vercel.app)
API Server (Backend):** [https://vuko-backend.onrender.com](https://vuko-backend.onrender.com)

## 📁 Estructura del Repositorio

[vuko-frontend/](./vuko-frontend): Cliente React + Vite.

[vuko-backend/](./vuko-backend): Servidor Node.js + MongoDB.

Postman Collection: Archivo `.json` en la raíz para pruebas de API.

## 🛠️ Tecnologías

MERN Stack (MongoDB, Express, React, Node.js), Vercel y Render.
